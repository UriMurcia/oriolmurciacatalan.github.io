Beat Repeat (WIP)

Videogame

My final degree project, in which I'm still working, is a 2D mobile application developed with Unity, where the user plays, creates and learns music at a very easy level. In this videogame there are 5 robots, each one with a different instrument (electric bass, electric guitar, keyboard, electric drums and singer). With a drag and drop system, the user is able to drag different melodies and drop them into each of these robots to play different combinations of music. Also the player is able to chose between 4 different sounds for each instrument and has the possibility to add different effects to each one of them. (Note: The robot designs are not mine and do not belong to the final product)
